Buy Monero Sell Monero API Documentation

API Documentation

Requests are rate-limited at 50 requests per second.

There is a 0.2% fee for all trades.

EndpointURL ParametersDescriptionExample Responses
/new_userGenerates a new API key.

/balances
  • api_key
Gets the balances for both XMR and BTC. 
{"XMR": "0E-18", "BTC": "0E-18"}
/orders
  • api_key
  • market
Gets a list of your open orders in market.

  "id":1,
  "price":"0.010000000000000000",

/trades
  • api_key
  • market
Gets a list of the trades you have made in market. 
[{
  "amount":"0.010000000000000000",
  "fee":"0.0000002000",
  "price":"0.010000000000000000",
  "timestamp": 1673138672,
  "type":"buy"
}]
/deposit
  • api_key
  • asset
Gets a deposit address for asset.
{"address": "bc1qh5jnz3tjtvththmd0sjru5sq97p8e9qj9mm794"}
/withdraw
  • api_key
  • asset
  • amount
 Withdraws amount of asset to address. Total withdrawn is amount-withdrawal fee.
{"success": true}
/buy
  • api_key
  • market
  • amount
  • price
 Creates an order to buy amount of base asset in market at price or below.
Returns the order_id, or success if the order was filled.
{"order_id": 1}
{"success": true}
/sell
  • api_key
  • market
  • amount
  • price
 Creates an order to sell amount of base asset in market at price or above.
Returns the order_id, or success if the order was filled.
{"order_id": 2}
{"success": true}
/cancel
  • api_key
  • order_id
Cancels an order.
{"success": true}
